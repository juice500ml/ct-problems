# 섭씨/화씨 변환기
****
### 설명
섭씨 온도를 입력받아서 화씨 온도로 변환하거나, 화씨 온도를 입력받아서 섭씨 온도로 변환하는 프로그램을 작성하시오. 다음 페이지의 프로그램 수행 결과를 보고 코드의 빈 부분을 채워서 프로그램을 완성하시오.

### 주의할 점
- 띄어쓰기도 체크하니 주의하세요.

- 시작 온도와 끝 온도는 정수만 입력합니다.
