# 피그 라틴
****
### 설명
피그 라틴(Pig Latin)은 어린이들의 말장난의 일종으로, **boy**를 **oybay**라 하는 것과 같이 맨 앞 자음을 어미로 붙이고 그 뒤에 **ay**를 추가하는 것이다. 하나의 단어를 입력받아서 피그 라틴으로 바꾸는 프로그램을 작성하시오.
예를 들면, happy → appy**h**ay, question → uestion**q**ay, love → ove**l**ay 등이 있다.

### 주의할 점
- 출력 형태에 맞게 해주세요.
- 단어는 두 글자 이상으로 들어옵니다.
