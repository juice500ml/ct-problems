# 팰린드롬 만들기
****
### 설명
AOA는 거꾸로 읽어도 AOA이다. 이처럼 **앞에서부터 읽으나 뒤에서부터 읽으나 동일한 단어나 구**를 팰린드롬(palindrome)이라고 한다. 임의의 단어를 입력받아 대칭을 이루도록 하여 팰린드롬을 만들어 출력하는 프로그램을 작성하라.

### 주의할 점
- 띄어쓰기도 체크하니 주의하세요.
- 단어에서 마지막 글자를 제외하고 나머지 글자들의 순서를 뒤집어 출력하면 회문이 됩니다.
